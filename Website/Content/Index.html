Meridiem

A CSC 4263 project created using Unity

Game Concept

Meridiem is 2D sidescrolling adventure game where the main character is on a mission to rescue their beloved life partner from your evil counterpart. The main character must travel across various worlds and learn new skills in order to get his/her beloved back.

Storyboard

Click on the image to view the storyboards our artists created for Meridian.

Game Mechanics

Meridian is a 2D sidescrolling adventure in which the player must utilize four unique power-ups in order to progress through the environment and ultimately rescue their significant other.
